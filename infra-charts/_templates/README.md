# \_templates

Шаблоны и скрипты для генерации новых чартов и поддержания единообразия в репозитории.

## Файлы

- `new-chart.sh` - интерактивный генератор нового Helm-чарта

## Использование

```bash
just new-chart <category> <name>;
# или напрямую
./_templates/new-chart.sh data clickhouse
```

## Критерий принадлежности

Компоненты в этой директории не деплоятся в кластер. Это инструменты для разработчиков репозитория.
